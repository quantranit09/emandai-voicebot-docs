# Create AI BOT

Start creating a new Voicebot AI as follows:

## Initialization

* Select "Create new virtual assistant" \[1]
* Name the Voicebot (Example: Customer survey Voicebot) \[2]
* Select AI Voicebot Type \[3]
* Select continue \[4] to the next step

## Method 1: Choose a Template bot

After completing step [(1) Create a new virtual assistant](https://docs.emandai.net/em-and-ai-voicebot-ai/tao-ai-bot#khoi-tao). In the display frame, select the built-in scenario in the list and edit the response.

### How to set up

* Select the scenario and listen to it \[1] (Select voice & press the button<img src=".gitbook/assets/image (12).png" alt="" data-size="line"> to listen)
* Select Continue \[2] to go to the parameter customization step.

<figure><img src=".gitbook/assets/image (80).png" alt=""><figcaption></figcaption></figure>

* Enter the required information needed in the scenario (Select button <img src=".gitbook/assets/image (23).png" alt="" data-size="line">to listen to the response script) \[3]
* Once set up, scroll down to the " Template bot" section to listen back to the entire scenario \[4]
* Select **Save** \[5] to complete.

<figure><img src=".gitbook/assets/image (81).png" alt=""><figcaption></figcaption></figure>

### Edit script

After creating a new Voicebot, select the newly created Voicebot and start editing the response as follows

(1) Select the newly created Voicebot \[1]

![](<.gitbook/assets/image (37).png>)

(3) Change the pre-built AI BOT Flow script.

* Change bot response content (BOT RESPONSE block).
* Add sample sentences and customer responses (CUSTOMER RESPONSE block).
* ...
