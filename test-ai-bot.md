# Test AI BOT

## Test chat <a href="#p9w5k6g6fy9n" id="p9w5k6g6fy9n"></a>

(1) Select the icon in the bottom corner of the screen \[1]

(2) The chat box displays, select version va-v2 \[2]

(3) Enter test response content in the chat box \[3]

{% hint style="info" %}
To reset the conversation, select the icon then select Reset conversation to start a new conversation
