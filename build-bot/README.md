# Build BOT

<figure><img src="../.gitbook/assets/image (796).png" alt=""><figcaption></figcaption></figure>

On the dashboard screen, you will see a list of chatbots that have been created in the system. To access a specific bot, simply click on it, and you will be redirected to the bot-building feature. This feature enables you to set up more advanced and detailed features for your ChatBot.

The menu bar on the left side of the screen shows the main features that you can set up for the Bot

<figure><img src="../.gitbook/assets/image (690).png" alt=""><figcaption><p>Main menu in building bot</p></figcaption></figure>

**Main menu:**

* **Bot settings:** This feature allows you to set some orientation settings in the bot control. Additionally, you can edit the displayed information and customize the chat interface more flexibly
* **Knowledge base:** Your chatbot will use the data you enter in this feature to analyze and build answers to customers' questions.
* **Demo AI Assistant:** simulate the chatbox interface and experience the interactive conversation flow with the bot
* **Advance features:**
  * **Task-oriented:** Advanced custom tasks can be set up for the bot, such as defining its response when customers mention specific predefined words, phrases, or sentences, and capturing information from their responses.
  * **Integration**: instruction on how to integrate chatBot on some platforms Facebook, Zalo, Website, Messager
  * **Usage details:** statistics of used parameters such as: the number of used tokens and number of documents
