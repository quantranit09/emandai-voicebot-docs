# Usage Details

**"Usage details"** is a statistical function that helps track the number of tokens used in detail by day/time, helping to control the number of tokens used by day/month, proactively making decisions based on actual user needs.

<figure><img src="../.gitbook/assets/image (287).png" alt=""><figcaption><p>Usage Details</p></figcaption></figure>

You can adjust the time period for tracking usage statistics by filtering (monthly/yearly)

<figure><img src="../.gitbook/assets/image (547).png" alt=""><figcaption><p>Filter</p></figcaption></figure>

**At the chart:**

\+ <mark style="color:blue;">**Blue:**</mark> represents the number of tokens used for training documents

<mark style="color:green;">**+ Green:**</mark> represents the number of tokens used for general use (including: bot testing, bot training, customers texting on the integrated site)

<figure><img src="../.gitbook/assets/image (548).png" alt=""><figcaption><p>Token classification</p></figcaption></figure>

**Daily Usage Breakdown**

You can view detailed usage information by adjusting the filter according to your desired viewing date

Includes 2 sections: Used Tokens and Training Tokens

<figure><img src="../.gitbook/assets/image (549).png" alt=""><figcaption><p>Used tokens</p></figcaption></figure>

The system will display the number of requests for each type of token, the time period in which the use occurred and the number of tokens used during that time.
