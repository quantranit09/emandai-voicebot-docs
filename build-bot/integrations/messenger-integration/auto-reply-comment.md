# Auto-reply comment

_<mark style="color:orange;">**Note:**</mark> <mark style="color:orange;"></mark><mark style="color:orange;">The feature only works if the user has integrated Messenger in the Integration tab</mark>_

## Add Task

**Step 1:** Click the **Reply comments** tab on the Messenger integration screen, the system displays a screen allowing comment reply settings.

<figure><img src="../../../.gitbook/assets/image (6).png" alt=""><figcaption><p>Click the Reply to comments tab to access</p></figcaption></figure>

**Step 2:** Click **Add** to create a new response on Facebook Messenger. The system displays a pop-up allowing you to fill in the name of the comment reply section. Enter a name and click **Save**, the system displays the newly created information on the comment reply list screen

<figure><img src="../../../.gitbook/assets/image (7).png" alt=""><figcaption><p>Click the <strong>Add</strong> button to proceed with adding information to reply to comments</p></figcaption></figure>

<figure><img src="../../../.gitbook/assets/image (8).png" alt=""><figcaption><p>Fill in the task name and click Add to add the task to the list</p></figcaption></figure>

**Step 3:** In the list of operations (showing articles under review), click to edit article information & feedback content.

<figure><img src="../../../.gitbook/assets/image (9).png" alt=""><figcaption><p>Click on the task to edit information</p></figcaption></figure>

**Step 4:** Optional posts need interaction, users can choose 3 options below

<figure><img src="../../../.gitbook/assets/image (10).png" alt=""><figcaption><p>Post options on Facebook page</p></figcaption></figure>

#### All&#x20;

When selecting this option, the system will check all posts on the page and respond (if the set conditions are met).

#### Contain hashtag&#x20;

When selecting this option, the system will check all posts containing the entered hashtag and respond (if the set conditions are met).&#x20;

**How to enter:** The users fill in the hashtag and press Enter on the keyboard, can enter an unlimited number of hashtags

<figure><img src="../../../.gitbook/assets/image (11).png" alt=""><figcaption></figcaption></figure>

#### Specific posts&#x20;

When selecting this option, the system will check for posts containing the entered ID and respond (if the set conditions are met).

<figure><img src="../../../.gitbook/assets/image (12).png" alt=""><figcaption></figcaption></figure>

**Step 5:** Enter keywords to check in the articles&#x20;

**How to enter:** The users fill in the keyword and press Enter on the keyboard, can enter an unlimited number of keywords to check

<figure><img src="../../../.gitbook/assets/image (13).png" alt=""><figcaption><p>Enter the keyword to check</p></figcaption></figure>

**Step 6:** Option to respond to comments and send messages

#### + Respond to comments&#x20;

When selecting this option, the system will respond to the comment according to the content the user has entered below (if the comment meets the established conditions).

#### + Send messages&#x20;

When selecting this option, the system will text the commenter according to the content the user has entered below (if the comment meets the established conditions).

<figure><img src="../../../.gitbook/assets/image (15).png" alt=""><figcaption><p>Option to respond to comments and send messages</p></figcaption></figure>

You can add feedback conditions for other keywords in the same article by pressing the **Add** button, the system displays a new tab allowing you to fill in keywords and other feedback content.

<figure><img src="../../../.gitbook/assets/image (16).png" alt=""><figcaption><p>Add response conditions for other keywords in the same article</p></figcaption></figure>

You can delete an added tab by clicking the **Remove this tab** button

<figure><img src="../../../.gitbook/assets/image (17).png" alt=""><figcaption><p>Click the <strong>Remove this tab</strong> button to delete tab</p></figcaption></figure>

**Step 7:** Click the **Save** button to save the information

<figure><img src="../../../.gitbook/assets/image (18).png" alt=""><figcaption><p>Click the <strong>Save</strong> button to save the information</p></figcaption></figure>
