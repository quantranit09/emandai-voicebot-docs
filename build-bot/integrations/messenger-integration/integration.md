# Integration

**Step 1:** The system will display the Messenger Integration tab -> click the **Connect to Messenger** button to proceed with the integration.

<figure><img src="../../../.gitbook/assets/image (2).png" alt=""><figcaption><p>Click the <strong>Connect to Messenger</strong> button to proceed with the integration</p></figcaption></figure>

Select your Facebook account to connect to VAgent

<figure><img src="../../../.gitbook/assets/image (3).png" alt=""><figcaption><p>Continue connecting using your current account or sign in to another account</p></figcaption></figure>

**Note:** the selected Facebook account must be an administrator of the Fanpage that needs to integrate the chatbot.&#x20;

**Step 2:** The system will display the fanpages of the connected Facebook account, you select the fanpage to integrate and click **Next**.

<figure><img src="../../../.gitbook/assets/image (4).png" alt=""><figcaption><p>Select the Fanpage to integrate -> click <strong>Next (Next)</strong></p></figcaption></figure>

**Step 3:** Set up access rights for VAgent, allowing Vagent to perform all of the tasks listed&#x20;

* Turn the Toggle to **Yes** to grant access to Vagent -> Click **Done**&#x20;

**Note:** EM\&AI VAgent will not be able to operate if the user does not grant permissions for these tasks

<figure><img src="../../../.gitbook/assets/image (5).png" alt=""><figcaption><p>Allow Virtual Agent access by clicking <strong>Yes</strong></p></figcaption></figure>

**Step 4:** The system displays the Messenger Integration pop-up, click Enable -> click Reload. You have now successfully integrated Virtual Agent into Facebook Messenger. You can access Facebook/Messenger to the integrated Fanpage conversation to check.



Thus, the steps to integrate into Facebook Messenger have been completed.&#x20;

If you want to set comment reply conditions, click the Comment Reply tab
