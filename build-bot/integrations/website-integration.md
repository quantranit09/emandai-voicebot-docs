# Website integration

**Step 1**: At the Integration screen, click on Website integration

<figure><img src="../../.gitbook/assets/image (790).png" alt=""><figcaption><p>click on Website integration</p></figcaption></figure>

**Step 2**: The website integration popup will display a code for you to copy and paste into your website.

<figure><img src="../../.gitbook/assets/image (791).png" alt=""><figcaption><p>Copy the code shown on the pop-up</p></figcaption></figure>

When you copy the code displayed on the pop-up, the Website Integration button will automatically activate. Click '**Save**'.

<figure><img src="../../.gitbook/assets/image (792).png" alt=""><figcaption><p>Activate the Website Integration button -> tap <strong>Save</strong></p></figcaption></figure>

**Step 3**: Paste the copied code into the website to be integrated&#x20;

* For WordPress use cases:&#x20;
  * Go to Settings → select Header and Footer Scripts
  *

      <figure><img src="../../.gitbook/assets/image (794).png" alt="" width="375"><figcaption><p>Go to Settings → select Header and Footer Scripts</p></figcaption></figure>
* Paste the copied code into the footer -> then click Save settings

<figure><img src="../../.gitbook/assets/image (795).png" alt=""><figcaption><p>Paste the copied code into the footer -> then click Save settings</p></figcaption></figure>

* In case you want to use another website, simply embed the copied script into the tag on your website.

Thus, the integration steps into the website are complete.
