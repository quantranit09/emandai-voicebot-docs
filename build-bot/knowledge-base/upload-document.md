# Upload Document

VAgent currently supports uploading documents, adjusting the content of documents or setting titles for different groups of documents with completely easy and fast operations.&#x20;

On the **Knowledge Base** screen, select **Upload File** to upload data using a file

<figure><img src="../../.gitbook/assets/image (57).png" alt=""><figcaption><p>Press Upload File button</p></figcaption></figure>

The system displays the File Upload popup, you can:&#x20;

* Press the arrow <img src="../../.gitbook/assets/image (58).png" alt="" data-size="line"> to select an available theme in the system&#x20;
* Or create a new topic by entering a new topic name in the information field![](<../../.gitbook/assets/image (59).png>)

<figure><img src="../../.gitbook/assets/image (60).png" alt=""><figcaption><p>Popup Upload file</p></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (61).png" alt=""><figcaption><p>Select the file to upload</p></figcaption></figure>

Select the file to upload with format requirements:&#x20;

* Format .pdf, .doc, .docx, .txt&#x20;
* Capacity less than 5MB&#x20;

After completing the topic selection and selecting the file to upload, select Upload to upload the selected file to the system.

<figure><img src="../../.gitbook/assets/image (62).png" alt=""><figcaption><p>Click Upload to upload the file</p></figcaption></figure>

VAgent allows review to edit uploaded documents

<figure><img src="../../.gitbook/assets/image (63).png" alt=""><figcaption><p>Preview uploaded documents</p></figcaption></figure>

Can see the number of documents to upload

<figure><img src="../../.gitbook/assets/image (64).png" alt=""><figcaption></figcaption></figure>

Edit another document by tapping <img src="../../.gitbook/assets/image (65).png" alt="" data-size="line"><img src="../../.gitbook/assets/image (66).png" alt="" data-size="line"> or clicking directly on the number containing the document

<figure><img src="../../.gitbook/assets/image (67).png" alt=""><figcaption><p>Click to select the document you want to edit</p></figcaption></figure>

You can fill in the title information for each document or apply the title to all uploaded documents by pressing **Apply to all**

<figure><img src="../../.gitbook/assets/image (68).png" alt=""><figcaption><p>Press Apply to all</p></figcaption></figure>

If you are satisfied with the content of the document and want to upload it to the system, click the **Train** button

<figure><img src="../../.gitbook/assets/image (69).png" alt=""><figcaption><p>Click Train to upload documents</p></figcaption></figure>

After pressing Train button, the system will conduct training documents. When training is completed, the system displays the message Training documents successfully

<figure><img src="../../.gitbook/assets/image (70).png" alt=""><figcaption><p>Notification of successful training</p></figcaption></figure>
