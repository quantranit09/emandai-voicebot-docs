# Document

Once you've created a topic, the next step is to provide content for it. This content will consist of several documents that contain the knowledge used to train the bot. The bot then analyzes and selects relevant information from these documents in order to provide answers to related questions.

Select and click on the specific topic, and the interface will display the screen to add documents to the topic

<figure><img src="../../.gitbook/assets/image (753).png" alt=""><figcaption><p>Interface to add documents for topic</p></figcaption></figure>

To add new documents, use the **'Add Document'** button at the top left of the screen to display the '**ADD DOCUMENT'** pop-up

<figure><img src="../../.gitbook/assets/image (754).png" alt=""><figcaption><p>'<strong>ADD DOCUMENT'</strong> pop-up</p></figcaption></figure>

Enter the name of your document

<figure><img src="../../.gitbook/assets/image (755).png" alt=""><figcaption><p>Click and enter a document name, or skip to get the default name</p></figcaption></figure>

The document should be named for convenience of storing and searching, with a short and accurate name.

* Please make sure to enter the required document content. This content will be used by the chatbot to analyze and develop responses to customers' queries. Therefore, it is important to enter the content in a comprehensive, detailed, and consistent manner. The more consistent and rich the data, the clearer and more accurate the Bot's response will be.&#x20;

<figure><img src="../../.gitbook/assets/image (756).png" alt=""><figcaption><p>Enter content of document</p></figcaption></figure>

The input content is limited to 1,400 tokens. The amount of tokens will depend on the account package that the user has registered

After entering the data content -> Click **Add** to save the created content

<figure><img src="../../.gitbook/assets/image (757).png" alt=""><figcaption><p>Click <strong>Add</strong> to save the created content</p></figcaption></figure>

After creating the document, the list of created documents will be displayed on the topic screen

<figure><img src="../../.gitbook/assets/image (758).png" alt=""><figcaption><p>Created document list</p></figcaption></figure>

The documents, after being created, will be automatically trained for the Bot.&#x20;

Only when the data is successfully trained can the bot use this data as a reference data source.

If there is an error while training, you need to re-train the bot by clicking the Retrain button<img src="../../.gitbook/assets/image (334).png" alt="" data-size="line">

**Edit document**

To edit the document, click on the Options button "..." to select the **Edit** document

<figure><img src="../../.gitbook/assets/image (759).png" alt=""><figcaption><p>Click on the Options button "..." to select the <strong>Edit</strong> document</p></figcaption></figure>

After click on **Edit** -> the system will display the **Edit Content** pop-up

<figure><img src="../../.gitbook/assets/image (760).png" alt=""><figcaption><p>Enter the content need to change -> click on <strong>Edit</strong> button to save changes</p></figcaption></figure>

\---

**Remove document**

<figure><img src="../../.gitbook/assets/image (761).png" alt=""><figcaption><p>Click on the Options button "..." to select the <strong>Remove</strong> document</p></figcaption></figure>

To delete the document, click on the Options button "..." to select **Remove** -> the system will display a pop-up to confirm whether you are sure to remove the topic or not

Since the removal cannot be undone, you must take caution to avoid data loss and time-consuming fixes.
