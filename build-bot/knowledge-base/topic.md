# Topic

The Knowledge base screen will show the list of topics created in the system

<figure><img src="../../.gitbook/assets/image (741).png" alt=""><figcaption><p>Created topic list</p></figcaption></figure>

The topic will be displayed with following information:&#x20;

* Created at: date time created topic
* Topic Name
* Content data: the number of created documents of the topic
* Status:&#x20;
  * <img src="../../.gitbook/assets/image (291).png" alt="" data-size="line">: activated (default state), the topic is used as training data for bots
  * <img src="../../.gitbook/assets/image (292).png" alt="" data-size="line">: de-activated, the topic is not used as training data for bots

Some actions that users can perform on this screen:&#x20;

* **Add topic**
* **Search for topic**
* **Edit topic**
* **Remove topic**&#x20;
* **Activate/De-activate topic**
* **Update documents for the topic**

\---

**Add topic**

Click on the **Add** button at the top left of the Knowledge Base screen, the system will display the **ADD TOPIC** pop-up

<figure><img src="../../.gitbook/assets/image (743).png" alt=""><figcaption><p>Click <strong>Add</strong> to add topic</p></figcaption></figure>

**Enter topic name:** The user should choose a short, precise, and clear name that represents the topic of data for easy storage and search.

Click the **Add** button to create a new topic, the newly created topic will appear on the screen

<figure><img src="../../.gitbook/assets/image (744).png" alt=""><figcaption><p>Created topic list</p></figcaption></figure>

\---

**Search for topic**

When there are multiple topics in the system, it can be challenging to find a specific topic by merely scanning through the list. However, you can search for the topic you want by typing its name in the search box. The system will then search the list and display the topic that matches the name you entered. This makes it easier to locate the desired topic without having to go through the entire list manually.

<figure><img src="../../.gitbook/assets/image (745).png" alt=""><figcaption><p>Enter content to search for topic</p></figcaption></figure>

\---

**Edit topic**

<figure><img src="../../.gitbook/assets/image (746).png" alt=""><figcaption><p>Click to select <strong>Edit</strong> from option menu</p></figcaption></figure>

To edit the topic, select the specified topic, in the '**Action**' column, click '...' -> select **Edit** -> the system will display a pop-up that allows you to edit the topic name

<figure><img src="../../.gitbook/assets/image (747).png" alt=""><figcaption><p>Enter changes of the new of topic -> click on Edit to save changes</p></figcaption></figure>

\---

**Remove topic**

<figure><img src="../../.gitbook/assets/image (748).png" alt=""><figcaption><p>Click to select <strong>Remove</strong> from option menu</p></figcaption></figure>

To remove a topic, select the specified topic, in the '**Action**' column, click '...' -> select **Remove**-> the system will display a pop-up confirming whether you are sure to remove the topic or not

<figure><img src="../../.gitbook/assets/image (749).png" alt=""><figcaption><p>Click on <strong>Confirm</strong> to remove topic</p></figcaption></figure>

Be cautious when removing data to avoid permanent loss and recovery.

\---

**Active/De-active topic**

Active/De-active topic: use/stop using topics as training data for bot.

There are two ways to activate/deactivate a specific topic:&#x20;

* You can perform right on the Knowledge base screen by click on the toggle button **'Activate/deactivate topic'**

<figure><img src="../../.gitbook/assets/image (750).png" alt=""><figcaption><p>Click on the toggle to de-active topic</p></figcaption></figure>

<img src="../../.gitbook/assets/image (291).png" alt="" data-size="line">: activated (default state), the topic is used as training data for bots

<img src="../../.gitbook/assets/image (292).png" alt="" data-size="line">: de-activated, the topic is not used as training data for bots

* Another way to take action is at the topic screen, you can click on the toggle button to Activate / de-activate the topic&#x20;

<figure><img src="../../.gitbook/assets/image (751).png" alt=""><figcaption><p>Click on the toggle to de-active topic</p></figcaption></figure>
