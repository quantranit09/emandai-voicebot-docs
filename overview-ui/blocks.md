# Blocks

Blocks are functional blocks that support conversation flow creation. The blocks are grouped by function as follows:

* [​Conversation Session](https://docs.emandai.net/em-and-ai-voicebot-ai/blocks/luot-hoi-thoai): Functional group that creates a response of AI BOT and USER
* ​Parameters: Function group that creates parameters and classification tags
* ​Trigger: The functional group creates action triggers according to conditions/intent for AI BOT
* [Other tools: ](https://docs.emandai.net/em-and-ai-voicebot-ai/blocks/khac)Functional groups that create conditions or end conversations.
