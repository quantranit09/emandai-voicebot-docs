# Tích hợp Website

**Bước 1:** Tại màn hình Tích hợp, nhấn chọn vào tích hợp **Website**

<figure><img src="../../.gitbook/assets/image (260).png" alt=""><figcaption><p>Nhấn chọn vào tích hợp Website</p></figcaption></figure>

Bước 2: Hệ thống sẽ hiển thị Pop-up Tích hợp Website -> sao chép và dán đoạn mã hiển thị trên pop-up vào trang website của bạn

<figure><img src="../../.gitbook/assets/image (262).png" alt=""><figcaption><p>Sao chép đoạn mã hiển thị trên pop-up</p></figcaption></figure>

Khi sao chép đoạn mã hiển thị trên pop-up -> hệ thống tự động kích hoạt nút tích hợp Website -> nhấn **Lưu**

<figure><img src="../../.gitbook/assets/image (264).png" alt=""><figcaption><p>Kích hoạt nút Tích hợp Website -> nhấn <strong>Lưu</strong></p></figcaption></figure>

**Bước 3:** Dán đoạn mã đã sao chép vào website cần tích hợp

* **Đối với các trường hợp sử dụng Wordpress:**&#x20;
  * Truy cập **Cài đặt** → chọn **Header and Footer Scripts**
  *

      <figure><img src="../../.gitbook/assets/image (265).png" alt="" width="361"><figcaption><p>Truy cập <strong>Cài đặt</strong> → chọn <strong>Header and Footer Scripts</strong></p></figcaption></figure>
  * Dán mã code đã sao chép vào phần footer -> sau đó nhấn **Lưu thay đổi**

<figure><img src="../../.gitbook/assets/image (266).png" alt=""><figcaption><p>Dán mã code đã sao chép vào phần footer -> sau đó nhấn <strong>Lưu thay đổi</strong></p></figcaption></figure>

* Trường hợp sử dụng Website khác, nhúng thẻ script đã sao chép ở trên vào thẻ tại trang web của bạn.

Như vậy, các bước tích hợp vào website đã hoàn tất.
