# Tích hợp (Integration)

<figure><img src="../../.gitbook/assets/image (518).png" alt=""><figcaption><p>Màn hình tích hợp chatbot</p></figcaption></figure>

Phần này sẽ hỗ trợ bạn tích hợp Chatbot vào một số nền tảng trực tuyến như:

* Zalo
* Website
* Messenger

Nhấn chọn vào nền tảng cụ thể bạn muốn tích hợp, hệ thống sẽ hiển thị pop-up thông tin, bạn cần phải điền một số thông tin cơ bản và một số cài đặt tùy chọn bổ sung để có thể tích hợp Chatbot đã tạo vào nền tảng trực tuyến của khách hàng
