# Livechat

To access Livechat, you need to select the virtual assistant you want to access

<figure><img src="../../.gitbook/assets/image (101).png" alt=""><figcaption><p>A list of virtual assistants</p></figcaption></figure>

Select the virtual assistant you want to access -> System displays **Overview** screen

<figure><img src="../../.gitbook/assets/image (102).png" alt=""><figcaption><p>Overview screen</p></figcaption></figure>

Click on **Livechat** -> The system displays Livechat screen

<figure><img src="../../.gitbook/assets/image (90).png" alt=""><figcaption><p>Livechat screen</p></figcaption></figure>

<mark style="color:red;">**(1)**</mark> Conversation flow in different channels, with:&#x20;

* <mark style="color:blue;">**Blue:**</mark> represents a new conversation that has not been clicked to view
* <mark style="color:green;">**Green:**</mark> shows that the conversation has been clicked to view&#x20;

<mark style="color:red;">**(2)**</mark> Conversations where you have not joined the room will be listed in the **AI Sales Assistant Chats** section&#x20;

<mark style="color:red;">**(3)**</mark> Conversations in which users have joined the room to chat directly with customers will be listed in the **My Chats** section.
