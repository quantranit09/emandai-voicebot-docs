# Thông báo (Broadcast)

> _Lưu ý:_ Phải tích hợp Facebook hoặc Zalo trước khi sử dụng tính năng

Để truy cập chức năng **Thông báo**, bạn cần chọn trợ lý ảo cần truy cập
