# Overview

To access the **Overview** function, you need to select the virtual assistant you want to access first

<figure><img src="../.gitbook/assets/image (95).png" alt=""><figcaption><p>A list of virtual assistants</p></figcaption></figure>

Select the virtual assistant you want to access -> The system displays **Overview** screen

<figure><img src="../.gitbook/assets/image (96).png" alt=""><figcaption><p>Overview screen</p></figcaption></figure>

Informations & Charts at Overview screen:

<figure><img src="../.gitbook/assets/image (97).png" alt=""><figcaption></figcaption></figure>

<mark style="color:red;">**(3)**</mark> View the number of customers (by date)
