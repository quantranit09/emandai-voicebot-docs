# Create new broadcast

You can create a new broadcast by clicking the **Create** button

<figure><img src="../../.gitbook/assets/image (915).png" alt=""><figcaption><p>Press Create to create a broadcast</p></figcaption></figure>

The system displays a popup allowing new creation. Fill full the information:&#x20;

* **Channel:** allows to select channel (Facebook/Zalo)&#x20;
* **Name:** allows to name the broadcast&#x20;
* **Fanpage:** allows to select the fanpage of the linked account (only available for Facebook channels)&#x20;
* **Last interaction date:** allows to select the date of the customer's last interaction in the channel&#x20;
* **Label:** allows selecting customers to send notifications by label&#x20;
* **Select response:** allows to choose the content of notification to customers&#x20;
* **Schedule:** allows you to choose the time to send notifications

<figure><img src="../../.gitbook/assets/image (916).png" alt=""><figcaption><p>Create Broadcast screen</p></figcaption></figure>

After filling in all information, click **Save** to create broadcast

<figure><img src="../../.gitbook/assets/image (917).png" alt=""><figcaption><p>Click <strong>Save</strong> to schedule broadcast on sending activities</p></figcaption></figure>
