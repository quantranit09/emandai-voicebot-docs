# Broadcast

> _Note:_ Have to integrate Facebook or Zalo before using the feature

To access **Broadcast**, you need to select the virtual agent you want to access

<figure><img src="../../.gitbook/assets/image (912).png" alt=""><figcaption><p>Virtual agent list screen</p></figcaption></figure>

Select the virtual agent you want to access -> System displays **Overview** screen

<figure><img src="../../.gitbook/assets/image (913).png" alt=""><figcaption><p>Overview screen</p></figcaption></figure>

Press the **Broadcast** button -> The system displays the Broadcast screen
