Package information that the user is registering to use includes the following information:&#x20;

* **Package information:** package name in use&#x20;
* **Expiry date:** displays the expiration date of the package the customer uses. If after the expiration date the customer does not renew, the customer's account will be transferred to the Trial account package.&#x20;
* **Tokens used:** number of tokens used in training data for the bot&#x20;
* **Documents:** the number of documents created in the training data for the bot

See details at General statistics

Upgrade package&#x20;

Click on Usage Overview to display the Usage Overview screen, describing the number of tokens, documents used as well as expiration information of registered packages.

To upgrade or register for more packages, click Upgrade -> the system will display the package selection screen. Users should upgrade the package to receive more advanced features, tokens and documents (see details). at Package Registration)
