# Package Subscribe

To subscribe to the package, you can:

* On the Dashboard page click **Upgrade**

<figure><img src="../../.gitbook/assets/image (691).png" alt=""><figcaption><p>Click Upgrade to navigate to the Package Selection screen</p></figcaption></figure>

* Or, on the Dashboard screen -> click on the user profile -> click **Change plan**

<figure><img src="../../.gitbook/assets/image (692).png" alt=""><figcaption><p>Click Change plan to navigate to the Package Selection screen</p></figcaption></figure>

Package selection screen interface

<figure><img src="../../.gitbook/assets/image (693).png" alt=""><figcaption><p>Click Monthly/Yearly to select the package subscription period by month/year</p></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (694).png" alt=""><figcaption><p>Information about the packages</p></figcaption></figure>

Each package will display the following information:&#x20;

* Package name&#x20;
* Fee amount calculated by month/year
* Parameters and features used

Click **Subscribe** at the selected package to make a transaction payment

<figure><img src="../../.gitbook/assets/image (31).png" alt=""><figcaption><p>Transaction information</p></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (32).png" alt=""><figcaption><p>Choose a subscription package</p></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (33).png" alt=""><figcaption><p>Select the number of months</p></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (34).png" alt=""><figcaption><p>Enter promote code (if any)</p></figcaption></figure>

In case the user wants to issue an invoice for this transaction, click **Issue an invoice** and fill in all information to issue an invoice, then click Buy Now. Invoices will be sent via the email address of the account making the payment

<figure><img src="../../.gitbook/assets/image (35).png" alt=""><figcaption><p>Click on Issue an invoice if you want to export an invoice</p></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (36).png" alt=""><figcaption><p>Fill in pieces of information to issue an invoice</p></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (699).png" alt=""><figcaption><p>Click Buy Now to make your payment</p></figcaption></figure>

When the transaction is successful, you will have additional usage amounts such as a number of tokens, documents,... according to the registered parameters.
