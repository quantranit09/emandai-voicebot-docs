To view account information, on the **Dashboard** page, click on the user profile -> select **Profile** / **Transactions** / **Change password** to be navigated to the corresponding tabs on the **Account** screen.

<figure><img src="../.gitbook/assets/image (716).png" alt=""><figcaption><p>Click to display account information</p></figcaption></figure>

<figure><img src="../.gitbook/assets/image (717).png" alt=""><figcaption><p>Account screen</p></figcaption></figure>

**+ Tab Profile**

<figure><img src="../.gitbook/assets/image (718).png" alt=""><figcaption><p>Tab Profile</p></figcaption></figure>

The **Profile** tab allows users to change information related to their account:&#x20;

* Upload an avatar
* Change account personal information such as:&#x20;
  * Full name
  * Phone number
  * Email
  * Address
  * Organization
* Information about the package being registered: users can upgrade their package by clicking the **Upgrade Plan** button

<figure><img src="../.gitbook/assets/image (719).png" alt=""><figcaption><p>Click <strong>Upgrade Plan</strong> to upgrade package</p></figcaption></figure>

**+ Tab Transaction**

<figure><img src="../.gitbook/assets/image (522).png" alt=""><figcaption><p>Transactions</p></figcaption></figure>

The **Transactions** screen displays the history of package registration transactions, including the following information:&#x20;

* Transaction date: date of completion of package registration and service fee payment. This is also the day to start calculating the validity period of the package&#x20;
* Package name with service unit&#x20;
* Price total payment&#x20;
* Payment code: payment transaction code&#x20;
* Payment method: payment method that the user has made: Internet banking, or payment by credit card...&#x20;
* Transaction time: execution time
