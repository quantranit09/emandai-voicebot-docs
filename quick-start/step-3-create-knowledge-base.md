# Step 3 - Create Knowledge base

After creating a chatbot, the next step is to add data to train the bot.&#x20;

In the menu on the left side of the screen â†’ select **Knowledge Base** to start generating the first data for the bot.&#x20;

The knowledge base will be organized into different topics, with each topic containing multiple documents submitted by users. This data will serve as a reference source for the bot to generate accurate answers to customer's questions.

<figure><img src="../.gitbook/assets/image (671).png" alt=""><figcaption><p>Knowledge base screen</p></figcaption></figure>

**Add Topic**

Click on the **Add** button at the top left of the Knowledge Base screen, the system will display the **ADD TOPIC** pop-up

<figure><img src="../.gitbook/assets/image (673).png" alt=""><figcaption><p>Click <strong>Add</strong> to add topic</p></figcaption></figure>

<figure><img src="../.gitbook/assets/image (674).png" alt=""><figcaption><p><strong>'ADD TOPIC'</strong> pop-up</p></figcaption></figure>

**Enter topic name:** The user should choose a short, precise, and clear name that represents the topic of data for easy storage and search.

Click the **Add** button to create a new topic, the newly created topic will appear on the screen

<figure><img src="../.gitbook/assets/image (675).png" alt=""><figcaption><p>Created topic list</p></figcaption></figure>

**Add document**

After creating a topic, the next thing to do is add data content for the topic you created.&#x20;

Select and click on the specific topic, and the interface will display the screen to add document to the topic

<figure><img src="../.gitbook/assets/image (676).png" alt=""><figcaption><p><strong>'Add document'</strong> screen</p></figcaption></figure>

To add documents, click on the '**Add Document'** button at the top left of the screen -> the system will display the '**ADD DOCUMENT'** pop-up

<figure><img src="../.gitbook/assets/image (679).png" alt=""><figcaption><p>'<strong>ADD DOCUMENT'</strong> pop-up</p></figcaption></figure>

* Enter a document name: give your document a name

<figure><img src="../.gitbook/assets/image (680).png" alt=""><figcaption><p>Tap to enter a document name, or skip to get the default name</p></figcaption></figure>

It is not mandatory to name documents, but for easy storage and retrieval, document names should be short, accurate, and memorable.

* **Enter content:** It is essential to enter complete and detailed information in this document as it will be used by the chatbot to analyze and provide answers to customers' inquiries. Consistency in the data entered is crucial for the bot to provide accurate responses. The more comprehensive and consistent the data, the clearer and more precise the bot's response will be. Therefore, please ensure to enter the information thoroughly and coherently to enhance the bot's effectiveness.

<figure><img src="../.gitbook/assets/image (681).png" alt=""><figcaption><p>Enter content of the document</p></figcaption></figure>

The data content is limited to 1,400 tokens. The amount of tokens will depend on the account package the user has registered

After entering the content -> Click '**Add'** button to save the created content

<figure><img src="../.gitbook/assets/image (682).png" alt=""><figcaption><p>Click '<strong>Add'</strong> button to save the created content</p></figcaption></figure>

After creating the document, the list of created documents will be displayed on the topic screen

<figure><img src="../.gitbook/assets/image (683).png" alt=""><figcaption><p>Created content list</p></figcaption></figure>

At the same time the document is created, the data will be automatically trained for the Bot.&#x20;

Only when the data is successfully trained, the bot can use this data as a reference data source.

<figure><img src="../.gitbook/assets/image (684).png" alt=""><figcaption><p>Interface showed that data was trained successfully</p></figcaption></figure>

If there is any error while training, you need to re-train the bot by clicking the Re-train button <img src="../.gitbook/assets/image (334).png" alt="" data-size="line">

**Edit document**

To edit the document, click on the Options button "..." to select the Edit/Remove document

<figure><img src="../.gitbook/assets/image (752).png" alt=""><figcaption><p>Click on the Options button "..." to select the Edit/Remove document</p></figcaption></figure>
