# NLP training

### NLP training <a href="#nxdxwmfw8etr" id="nxdxwmfw8etr"></a>

NLP training tool helps AI BOT understand customer statements by detecting intent, thereby providing responses corresponding to the conversation flow.

<figure><img src="../.gitbook/assets/image (76).png" alt=""><figcaption></figcaption></figure>

**How to set up**

(1) Select button <img src="../.gitbook/assets/image (46).png" alt="" data-size="line"> \[1] to add a new intent

(2) Add new intent as follows:

* Intent name (Example: greeting, farewell,...) \[2]
* Add sample sentences that represent intent \[3]
* Keywords are used to train AI BOT to recognize words of the same type, abbreviations, borrowed words, synonyms... that have the same meaning as a certain word. \[4]
* Select Save to complete \[5]

(3) Select button <img src="../.gitbook/assets/image (17).png" alt="" data-size="line"> \[6] to train AI BOT to learn the sample sentences and intents just created.

<figure><img src="../.gitbook/assets/image (77).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
NLP training is needed after each step of adding/editing/deleting intent or sample sentences, and keywords.
{% endhint %}

### NLP Test

Select the NLP Test function, NLP Test shows the model's recognition results after training:

* Select button<img src="../.gitbook/assets/image (19).png" alt="" data-size="line"> \[1] to test NLP
* Enter the content of the sample sentence to be tested \[2]
* Select Analyze \[3] for NLP to process. Select Cancel to exit

<figure><img src="../.gitbook/assets/image (78).png" alt=""><figcaption></figcaption></figure>
