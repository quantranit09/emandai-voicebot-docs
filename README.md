# Document Overview

### Purpose of the document <a href="#be0xo2gy750u" id="be0xo2gy750u"></a>

The document aims to introduce an overview of Bot Flow and guide users to effectively use the conversation design functions for AI VOICEBOT.

### Subjects of use <a href="#etgxxrmqn6vm" id="etgxxrmqn6vm"></a>

Voicebot conversation design specialist: Build and manage Voicebot.

This new version - VA version 2 - BOT Flow script, is suitable for building and managing Voicebot to perform basic call scripts. For more complex call scripts, EM\&AI encourages users to use the previous version.- VA version 1 - Context scenario.
