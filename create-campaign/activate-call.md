# Activate call

After creating the conversation scenario and NLP training, set the following parameters to activate calls for Voicebot.

<figure><img src="../.gitbook/assets/image (87).png" alt=""><figcaption></figcaption></figure>

**How to set up**&#x20;

(1) Select Integration

(2) Select Voicebot

(3) Set the following parameters:

* Select voice type: Select voice for AI BOT in the list. Then select the button
*   Version: Select Version\_2 to use the conversation flow script you just created

    &#x20;<img src="../.gitbook/assets/image (41).png" alt="" data-size="line"> to listen.

<figure><img src="../.gitbook/assets/image (88).png" alt=""><figcaption></figcaption></figure>

* Speaking speed: Choose the appropriate reading speed for AI BOT
* Number format: Default / +84xxx / 0xxx
* Switch device (if any)
  * Profile (Name of operator/department)
  * Ext (Name / Extension number)

<figure><img src="../.gitbook/assets/image (89).png" alt=""><figcaption></figcaption></figure>

(4) Select "Save" to complete
